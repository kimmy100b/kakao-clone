# 카카오톡 클론

## 소개

html과 css를 사용하여 카카오톡과 똑같이 만들어보는 클론 코딩입니다.
<br>

## 개발기간

2020.08.18 - 2020.08.31(14일)
<br>

## 개발인원

1명
<br>

## 핵심기능

- 로그인 페이지
- 친구리스트 페이지
- 채팅 페이지
- 채팅방 페이지
- 찾기 페이지
- 더보기 페이지

<br>
